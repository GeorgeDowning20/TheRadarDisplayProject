This files contains the main function to test the sixth and seventh subtasks of the project. the main function creates and instance of an empty icon and then it is initialised as a diagonal line using the #initialiseAsDefaultDiagonalLine(uint8_t Brightness) function from the Icon class. the icon is then printed to the console using the #showIconDetail function from the Icon class.

Doxygen Documentation for project: https://georgedowning20.github.io/TheRadarDisplayProject/index.html

#include "../Radar/Pixel/Pixel.cpp" // Pixel.cpp

// Main Program
//---------------------------------------------------------------------------//
int main()
{
icon myIcon; // Create an icon object
if (myIcon.initialiseAsDefaultDiagonalLine(15) != STATUS_OK) // If the icon is not initialised successfully
{
std::cout << "Error: Icon not initialised successfully" << std::endl; // Print an error message
return STATUS_ERROR;
}
if (myIcon.showIconDetail() != STATUS_OK) // If the icon detail is not shown successfully
{
std::cout << "Error: Icon detail not shown successfully" << std::endl; // Print an error message
return STATUS_ERROR;
}
}
