This file contains a main function to test the third subtask of the project. the main functions creates an instance of a pixel with the specified input data. the characteristics of the pixel are then printed to the console using the #showPixelDetail(aPixel) function.

Doxygen Documentation for project: https://georgedowning20.github.io/TheRadarDisplayProject/index.html

Example:
//---------------------------------------------------------------------------//
//---------------------------------------------------------------------------//
//---------------------------------------------------------------------------//
// preprocessor directives
//---------------------------------------------------------------------------//
#include "../Radar/Pixel/Pixel.cpp" // Pixel.cpp
//---------------------------------------------------------------------------//
// Main Program
//---------------------------------------------------------------------------//
int main()
{
pixel aPixel(3.7, 4.1, 7); // Create a pixel object with the specified input data: x and y coordinates to 3.9 and 4.1 and its brightness to 7.
if (pixel::showPixelDetail(aPixel) != STATUS_OK) // If the pixel detail is not shown successfully
{
std::cout << "Error: Pixel detail not shown successfully" << std::endl; // Print an error message
}
return 0; // Return 0
}
