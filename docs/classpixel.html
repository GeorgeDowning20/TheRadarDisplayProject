Detailed Description

This class represents a single pixel.

This class represents a single pixel and contains all the information required to represent a pixel. This includes the x and y coordinates and the brightness of the pixel. This class also contains functions to show the details of the pixel. This class is used by the icon class to represent the pixels in an icon.
